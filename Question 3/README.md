# Weekly Temperature Tracker

A straightforward Python program that collects a week's worth of temperature data and calculates key statistics.

## How It Works
When executed, the script performs the following actions:
1. **Data Entry:** Uses a loop to prompt the user to input a temperature value for 7 consecutive days, storing each entry in a list.
2. **Calculation:** Analyzes the list using Python's built-in functions to compute the highest, lowest, total sum, and average of the recorded temperatures.
3. **Output:** Displays the final statistics to the user, neatly formatted to two decimal places for clean readability.
