# Stone Knife Paper (SKP) Game

A simple command-line Python game where a player competes against the computer. 

## How It Works
When executed, the program prompts the user to input their choice:
* **S** for Stone
* **K** for Knife
* **P** for Paper

The computer randomly selects its own choice. [cite_start]The winner is then determined and printed to the console based on the core game rules:
* **Stone** blunts Knife
* **Knife** cuts Paper
* **Paper** wraps Stone

If the user's input equals the computer's choice, no one wins and a draw is declared.
